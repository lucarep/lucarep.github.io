5_A : ‘SAY WHAT AGAIN’

2021-10-10 :: Luca
Intro in this task, I’ve realized a program that reads as input a text file (.txt), shows its content and computes the 25 most frequent words inside, showing them in an appropriate histogram. I have purposely chosen to show only the first 25 most frequent words, because otherwise it would have been difficult to properly display all the words present, which can be many more. However, you can still see a full dictionary of the words with their relative frequency, just below the text.
